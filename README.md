# ThatsApp

Eine Android-App (Kotlin + Jetpack Compose) fuer Chat-Funktionen mit Fokus auf mobile Features wie Kamera, GPS, Dateiablage und MQTT-Kommunikation.

## Voraussetzungen

- Android Studio (aktuelle Version)
- JDK 17
- Android SDK gemaess Gradle-Konfiguration

## Projekt starten

```bash
./gradlew assembleDebug
```

Oder in Android Studio direkt ein Emulator-/Geraete-Target auswaehlen und starten.

## Projektstruktur

- `app/src/main/kotlin/fhnw/emoba/thatsapp`: App-Logik, Datenmodelle und UI
- `app/src/main/res`: Ressourcen (Strings, Themes, XML)
